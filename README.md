Skillmetter is a web platform that compares two TryHackMe user profiles and generates detailed skill-based comparisons, visual analytics, and an overall winner verdict with custom cyber titles.

## âœ¨ Features

### Core Features
- **Profile Comparison System** - Enter two TryHackMe usernames and initiate a skill comparison
- **Comprehensive Analysis** - Compare ranks, rooms, badges, certifications, streaks, and more
- **Domain Intelligence** - Analyze performance across 13 cyber security domains
- **Weighted Scoring** - Fair comparison using a weighted scoring algorithm
- **Cyber Titles** - Earn custom titles like "Red Team Predator" or "Blue Team Sentinel"
- **Visual Analytics** - Radar charts, bar graphs, and progress indicators

- Node.js 18+

### Installation

```bash

cd Skillmetter

# Install dependencies
npm install

# Start development server

```

### Build for Production

